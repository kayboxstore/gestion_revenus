# Guide utilisateur

Depuis l'accueil, consultez chiffre d'affaires, bénéfice brut, bénéfice net, dépenses familiales, épargne et trésorerie. Le bouton Ajouter permet les ventes, dépenses, transferts, apports, contributions d'épargne et achats de stock. Les activités IPTV, Mini UPS et Android TV Box sont actives; Billard reste inactif jusqu'à activation.
