# Gestion des revenus — Famille Kay

MVP Next.js/Supabase mobile-first pour piloter revenus, dépenses, transferts, stock, épargne et rentabilité d'une petite famille sans confondre trésorerie et revenu.

## Démarrage local

```bash
npm install
npm run dev
```

Copier `.env.example` vers `.env.local` pour connecter Supabase.

## Validation

```bash
npm run lint
npm run typecheck
npm run test
npm run test:integration
npm run build
npm run test:e2e
```
